# 💭 Idées en vrac

> [!info]
> Espace libre — toutes les idées s'ajoutent ici sans filtre. On trie après.

## 🎬 Produit

- [ ] Mode « vinyle numérique » : animation d'ouverture d'un boîtier DVD/Blu-ray virtuel quand on lance un film
- [ ] Collection « Criterion-like » — éditions spéciales avec bonus, commentaires, director's cut
- [ ] Badge NFT commémoratif pour les premiers acheteurs d'un film (collector)
- [ ] Playlists de films thématiques créées par cinéphiles connus, achetables en bundle
- [ ] Intégration Letterboxd : import watchlist, notation
- [ ] Mode « cinéma privé » : watch party synchrone P2P entre amis
- [ ] Statistiques personnelles : temps écran, films vus, bilan annuel (à la Spotify Wrapped)

## 🛒 Store

- [ ] Précommande de films qui sortent en salle (téléchargement dispo au jour de sortie)
- [ ] Bundles "Trilogie X" -30%
- [ ] Wishlist publique partageable
- [ ] Cadeaux : offrir un film à un proche (certificat transféré)
- [ ] Option "Tangible Pass fondateur" pour les early adopters — tarif préférentiel à vie

## ♻️ Marché secondaire

- [ ] Cap dynamique sur le prix de revente (éviter spéculation sur raretés)
- [ ] Auction-style pour éditions limitées
- [ ] Historique public anonymisé des ventes par film (transparence marché)

## 🌱 Numérique responsable

- [ ] Publier un **bilan CO2** annuel par utilisateur (streaming évité vs. téléchargement)
- [ ] Partenariat avec une asso de reforestation (1 film acheté = 1 arbre ?)
- [ ] Politique de transparence totale (open finances, open specs)

## 🤝 Partenariats potentiels

- [ ] **La Cinémathèque Française** — catalogue patrimonial en restauration 4K
- [ ] **Festivals** : Cannes, Deauville, Clermont-Ferrand — sorties directes post-festival
- [ ] **CNES cinéma d'animation**
- [ ] **UGC / Pathé** : partenariat pour proposer l'achat digital de films sortis en salle
- [ ] **BPI France** pour l'amorçage

## 🎮 Extensions marché

- [ ] Musique : appliquer le même modèle aux albums ?
- [ ] Livres numériques : propriété réelle d'un ebook ?
- [ ] Jeux vidéo indé (similarité avec GOG) ?

## 🪄 Branding / Marque

- [ ] Manifeste public type « Déclaration de propriété numérique »
- [ ] Sticker « I own my films » en goodies
- [ ] Podcast cinéphile sponsorisé / produit par Tangible
- [ ] Festival Tangible (édition virtuelle) chaque année
